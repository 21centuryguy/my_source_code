NAME
    markdown.extensions.wikilinks

DESCRIPTION
    WikiLinks Extension for Python-Markdown
    ======================================
    
    Converts [[WikiLinks]] to relative links.
    
    See <https://Python-Markdown.github.io/extensions/wikilinks>
    for documentation.
