DESCRIPTION
    Usage::
    
        >>> from lxml.html.builder import *
        >>> html = HTML(
        ...            HEAD( TITLE("Hello World") ),
        ...            BODY( CLASS("main"),
        ...                  H1("Hello World !")
        ...            )
        ...        )

          </head>
          <body class="main">
